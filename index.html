Projetos de Análise de Dados

Projetos em Python

Desenvolvimento de um Painel Gerencial para Negócio com Python e Streamlit

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimento web com o Streamlit e Github, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de delivery de comida na Índia.

O resultado final do projeto foi um painel hospedado em um ambiente Cloud e disponibilizado através de um link web. O painel pode ser acessado por qualquer dispositivo conectado na internet.

As ferramentas utilizadas foram:

  • Python.
  • Jupyter Notebook.
  • Visual Studio Code.
  • Terminal.
  • Streamlit.
  • Streamlit Cloud.
  • Github.

Dashboard De Vendas

O intuito do dashboard é monitorar o desempenho das vendas de uma distribuidora especializada em eletroeletrônicos no período de 2017 a 2019, acompanhar o total vendido, o lucro total, margem percentual, total vendido por continente e comparação de vendas entre os anos. O dashboard foi desenvolvido utilizando SQL Server e o design do dashboard feito com o Figma. Clique na imagem para visualizar.
